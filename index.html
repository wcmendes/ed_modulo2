📚 Conceitos Fundamentais

O que é um Registro?

Um registro (struct) é uma estrutura de dados heterogênea que permite agrupar diferentes tipos de dados relacionados sob um único nome, formando uma unidade lógica coesa.

Vetor vs Vetor de Registros

Analogia: Pense em uma planilha - cada linha é um registro, cada coluna é um campo. Um vetor simples seria apenas uma coluna, enquanto um vetor de registros é a planilha completa.

Boas Práticas

  • Nomeação clara: Use nomes descritivos para entidades e atributos
  • Modularização: Separe responsabilidades (validação, CRUD, UI)
  • Validação: Sempre valide tipos e restrições

Quando usar Registros?

Use registros quando precisar agrupar dados relacionados que pertencem logicamente à mesma entidade, em vez de usar múltiplas variáveis ou vetores separados.

📋 Registros (Structs)

            
            

                

                    
3.1. Estrutura e Componentes

                    
Um registro é composto por um conjunto de campos (também chamados de membros ou atributos), onde cada campo possui um nome único e um tipo de dado específico. A beleza dos registros reside na sua capacidade de combinar dados de naturezas distintas em uma única entidade.


                    

                        
Características dos Registros:

                        
    
                            
  • Heterogeneidade: Permitem armazenar dados de diferentes tipos em uma única estrutura.
    • 
                            
  • Organização Lógica: Agrupam informações que pertencem logicamente a uma mesma entidade.
    • 
                            
  • Acesso por Nome: Os campos são acessados pelo seu nome, melhorando a clareza do código.
    •

3.3. Aplicações e Importância

            

                

                    
🗄️ Bancos de Dados

                    
Representam linhas em tabelas, onde cada campo do registro corresponde a uma coluna da tabela.

                

                

                    
🎯 Programação Orientada a Objetos

                    
São a base para a criação de classes e objetos, onde os campos do registro se tornam os atributos da classe.

                

                

                    
🔗 Estruturas Complexas

                    
Podem ser combinados com outras estruturas, como vetores de registros ou listas encadeadas.

                

                

                    
🔄 Troca de Dados

                    
São ideais para empacotar informações que precisam ser passadas entre diferentes partes de um programa.

                

            

        


        
        

            
📊 Vetores de Registros

            
            

                

                    
A combinação de vetores e registros é uma técnica poderosa para organizar coleções de dados complexos. Um vetor de registros é essencialmente um array onde cada elemento é um registro, permitindo armazenar múltiplas instâncias de uma estrutura de dados heterogênea.


                    
Operações Comuns

                    

                        
    
                            
  • Inserção: Adicionar um novo registro ao vetor.
    • 
                            
  • Busca: Encontrar um registro específico com base no valor de um de seus campos.
    • 
                            
  • Remoção: Excluir um registro do vetor.
    • 
                            
  • Atualização: Modificar os valores dos campos de um registro existente.
    • 
                        

                    


                    
Aplicações

                    

                        

                            
💼 Sistemas de Gerenciamento

                            
Para armazenar e manipular coleções de objetos ou entidades.

                        

                        

                            
🏢 Aplicações Empresariais

                            
Gerenciamento de clientes, produtos, pedidos, etc.

                        

                        

                            
🎮 Simulações

                            
Modelagem de múltiplos agentes ou elementos com propriedades complexas.

from dataclasses import dataclass
from datetime import date
from typing import List

@dataclass
class Estudante:
    nome: str
    matricula: int
    curso: str
    media_geral: float
    data_nascimento: date

# Vetor de registros
estudantes: List[Estudante] = []

# Adicionando estudantes
estudantes.append(Estudante(
    nome="Maria Silva",
    matricula=12345,
    curso="Engenharia de Software",
    media_geral=8.75,
    data_nascimento=date(2003, 5, 10)
))

estudantes.append(Estudante(
    nome="João Souza",
    matricula=67890,
    curso="Ciência da Computação",
    media_geral=9.2,
    data_nascimento=date(2002, 11, 20)
))

# Buscar estudante por matrícula
def buscar_por_matricula(matricula: int) -> Estudante:
    for estudante in estudantes:
        if estudante.matricula == matricula:
            return estudante
    return None

# Listar todos ordenados por média
estudantes_ordenados = sorted(estudantes, 
                            key=lambda x: x.media_geral, 
                            reverse=True)

for estudante in estudantes_ordenados:
    print(f"{estudante.nome}: {estudante.media_geral}")
                            

                        

                        
                        

                            
import java.time.LocalDate;
import java.util.*;

public class GerenciadorEstudantes {
    private List<Estudante> estudantes;
    
    public GerenciadorEstudantes() {
        this.estudantes = new ArrayList<>();
    }
    
    // Adicionar estudante
    public void adicionarEstudante(Estudante estudante) {
        estudantes.add(estudante);
    }
    
    // Buscar por matrícula
    public Estudante buscarPorMatricula(int matricula) {
        for (Estudante estudante : estudantes) {
            if (estudante.getMatricula() == matricula) {
                return estudante;
            }
        }
        return null;
    }
    
    // Listar ordenados por média
    public List<Estudante> listarPorMedia() {
        List<Estudante> copia = new ArrayList<>(estudantes);
        copia.sort((e1, e2) -> 
            Double.compare(e2.getMediaGeral(), e1.getMediaGeral()));
        return copia;
    }
    
    // Remover estudante
    public boolean removerPorMatricula(int matricula) {
        return estudantes.removeIf(e -> e.getMatricula() == matricula);
    }
    
    public static void main(String[] args) {
        GerenciadorEstudantes gerenciador = new GerenciadorEstudantes();
        
        gerenciador.adicionarEstudante(new Estudante(
            "Maria Silva", 12345, "Eng. Software", 8.75,
            LocalDate.of(2003, 5, 10)
        ));
        
        gerenciador.adicionarEstudante(new Estudante(
            "João Souza", 67890, "Ciência da Computação", 9.2,
            LocalDate.of(2002, 11, 20)
        ));
        
        // Buscar e exibir
        Estudante encontrado = gerenciador.buscarPorMatricula(12345);
        if (encontrado != null) {
            System.out.println("Encontrado: " + encontrado.getNome());
        }
    }
}
                            

                        

                        
                        

                            
#include <stdio.h>
#include <string.h>
#include <stdlib.h>

#define MAX_ESTUDANTES 100

struct Estudante {
    char nome[100];
    int matricula;
    char curso[100];
    float media_geral;
    char data_nascimento[11];
};

struct VetorEstudantes {
    struct Estudante estudantes[MAX_ESTUDANTES];
    int total;
};

// Adicionar estudante
void adicionar_estudante(struct VetorEstudantes* vetor, 
                        struct Estudante estudante) {
    if (vetor->total < MAX_ESTUDANTES) {
        vetor->estudantes[vetor->total] = estudante;
        vetor->total++;
    }
}

// Buscar por matrícula
struct Estudante* buscar_por_matricula(struct VetorEstudantes* vetor, 
                                      int matricula) {
    for (int i = 0; i < vetor->total; i++) {
        if (vetor->estudantes[i].matricula == matricula) {
            return &vetor->estudantes[i];
        }
    }
    return NULL;
}

// Listar todos
void listar_estudantes(struct VetorEstudantes* vetor) {
    printf("Lista de Estudantes:\n");
    for (int i = 0; i < vetor->total; i++) {
        printf("%d. %s (Mat: %d) - Média: %.2f\n", 
               i+1, 
               vetor->estudantes[i].nome,
               vetor->estudantes[i].matricula,
               vetor->estudantes[i].media_geral);
    }
}

int main() {
    struct VetorEstudantes vetor = {.total = 0};
    
    // Criar estudantes
    struct Estudante e1 = {"Maria Silva", 12345, "Eng. Software", 8.75, "2003-05-10"};
    struct Estudante e2 = {"João Souza", 67890, "Ciência da Computação", 9.2, "2002-11-20"};
    
    // Adicionar ao vetor
    adicionar_estudante(&vetor, e1);
    adicionar_estudante(&vetor, e2);
    
    // Listar todos
    listar_estudantes(&vetor);
    
    // Buscar específico
    struct Estudante* encontrado = buscar_por_matricula(&vetor, 12345);
    if (encontrado != NULL) {
        printf("Encontrado: %s\n", encontrado->nome);
    }
    
    return 0;
}
